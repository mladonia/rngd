rngd

rngd and rngcli are a pair of tools to provide secure random number generation. rngd can be easily modified to support any modern secure prng-alghorithm.

usage

after installing rngcli, change the HOST variable to point to the application server. typical usage scenario:

./rngcli new > token
cat token | ./rngcli next > random_bytes

use cat token | ./rngcli entropy to view inner state of the generator.
